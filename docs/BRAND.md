# Brand

## Name

The product name is **SpacePilot**.

Use **SpacePilot** as one word with capital S and P.

Recommended descriptor:

```text
SpacePilot
Safe cleanup and optimization for Windows.
```

## Positioning

SpacePilot should feel precise, calm, and trustworthy. It helps users reclaim space and control without making reckless automated changes.

## Logo

The logo mark combines:

- A shield-like form for safety.
- A glass pane for system clarity.
- A sweeping arc for cleanup and reclaimed space.
- A check/spark detail for verified maintenance.

Editable source:

```text
src\SpacePilot\Assets\AppLogoMark.svg
```

Generated Windows icon:

```text
src\SpacePilot\Assets\AppIcon.ico
```

Regenerate icon assets with:

```powershell
node .\scripts\generate-brand-assets.mjs
```

## Naming Guidelines

Use:

- SpacePilot
- SpacePilot for Windows
- SpacePilot cleanup receipt
- SpacePilot quarantine

Avoid:

- Windows Cleaner as a product name.
- Microsoft or Windows logo-style marks.
- Claims that imply guaranteed performance improvements.
- Phrases like "registry repair" or "one-click fix everything."
